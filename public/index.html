How this site works

Each tool runs entirely in your browser and is built to answer one question well. Where a calculation involves choices — tax treatment, margin type, fees — the page says which choice it made and why, so the number is auditable. This is deliberately the opposite of a tool that hands you a figure and hopes you don’t ask how it got there.
